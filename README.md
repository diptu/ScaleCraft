# ScaleCraft 🛠️  
**A hands-on system design lab for scalable web and ML systems**

---

## 📌 Overview

**ScaleCraft** is a practical system design practice repository focused on designing, building, and scaling real-world systems.  
This repo bridges **system design theory** with **production-grade implementations** using modern web and ML stacks.

The goal is simple:

> **Design like an architect. Build like an engineer. Scale like a production system.**

---

## 🎯 Objectives

- Practice **end-to-end system design** (requirements → architecture → implementation)
- Implement scalable backends using **FastAPI**
- Build user-facing apps using **MERN**
- Design and serve **ML models** using **PyTorch**
- Apply real-world concepts: caching, sharding, async processing, queues, and observability
- Prepare for **system design interviews** and **senior backend roles**

---

## 🧰 Tech Stack

### Backend
- **FastAPI** – High-performance API services
- **Python** – Core backend and ML glue
- **PostgreSQL / MongoDB** – Data storage
- **Redis** – Caching & rate limiting
- **Celery / Background Workers** – Async processing

### Frontend
- **MongoDB**
- **Express.js**
- **React**
- **Node.js**

### Machine Learning
- **PyTorch** – Model development
- **ONNX / TorchScript** – Model optimization
- **FastAPI ML Inference Services**

### Infrastructure & DevOps
- **Docker & Docker Compose**
- **Nginx**
- **CI/CD (GitHub Actions)**
- **Cloud-ready design (AWS / Azure)**
